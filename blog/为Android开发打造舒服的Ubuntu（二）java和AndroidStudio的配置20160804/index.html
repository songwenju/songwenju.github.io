上一篇博文介绍了Ubuntu的安装和配置，以及一些神器的安装，这一篇介绍java和Android Studio的配置。

一，下载java和配置java环境变量

参考下面教程：

http://www.linuxidc.com/Linux/2015-01/112030.htm

二，下载Android Studio及Sdk

下载地址：http://www.androiddevtools.cn/

下载完毕后，解压安装到相应的位置。

三， 为Android studio添加启动图标

AS在linux不能直接单击打开，需要用打开shell脚本的方式打开，在终端输入：. studio.sh

这里配置启动图标。可以搜索（点击键盘的windows键），打开后可以把它固定在左侧启动栏，下次直接点击打开。

1）位置：/usr/share/applications

2）新建文件：studio.desktop,将下列配置文件拷入并修改studio.sh的目录和图标的目录：

[Desktop Entry]             

  Encoding=UTF-8

 Name=Android Studio

 Comment=Android studio

 Exec=/home/songwenju/Documents/develop/as/android-studio/bin/studio.sh   （修改成你自己的）

  Icon=/home/songwenju/Documents/develop/as/android-studio/bin/studio.png（修改成你自己的）

 Terminal=false

 StartupNotify=true

 Type=Application

 Categories=Application;Development;

然后搜索可以看到studio的图标。
