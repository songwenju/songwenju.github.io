三， 为Android studio添加启动图标

AS在linux不能直接单击打开，需要用打开shell脚本的方式打开，在终端输入：. studio.sh

这里配置启动图标。可以搜索（点击键盘的windows键），打开后可以把它固定在左侧启动栏，下次直接点击打开。

1）位置：/usr/share/applications

2）新建文件：studio.desktop,将下列配置文件拷入并修改studio.sh的目录和图标的目录：

[Desktop Entry]             

  Encoding=UTF-8

 Name=Android Studio

 Comment=Android studio

 Exec=/home/songwenju/Documents/develop/as/android-studio/bin/studio.sh   （修改成你自己的）

  Icon=/home/songwenju/Documents/develop/as/android-studio/bin/studio.png（修改成你自己的）

 Terminal=false

 StartupNotify=true

 Type=Application

 Categories=Application;Development;

然后搜索可以看到studio的图标。

为Android开发打造舒服的Ubuntu（一）Ubuntu安装配置及神器的安装

发表于 2016-08-02   |   分类于 Linux   |  

正儿八经用Ubuntu已经快一年了，这一年从熟悉Linux的基本命令到学写简单的脚本，越来越发现已经喜欢上它了。方便之处比如说进入某个层级很深的目录或用adb连接开发板等，用一个简单的shell脚本，然后敲一个单词就搞定了，对于shell脚本可以以后来展开。今天写这一篇博客是因为我的电脑升级了，不仅是主机升级，内存也从4G翻了4倍到了16G，用起来心里美多了，这个时候就需要重新的安装和配置Ubuntu系统，在这里记录和分享一下配置的过程。

一.U盘安装和系统更新

1.下载Ubuntu的.iso文件,可以在Ubuntu官网上下载。

2.制作u盘启动盘。参考下面教程：

http://jingyan.baidu.com/article/b24f6c82cf50e086bfe5dae9.html

3.根据不同的电脑设置开机为U盘启动，我用的是lenovo的主机，重启按f12进入选择启动方式，选择u盘启动。

4.安装过程参考下面教程：

http://jingyan.baidu.com/article/a378c9609aaf4eb3282830e6.html

注意：安装完毕后，要拔掉u盘后重启，否则重启后会继续进入安装阶段。

5.系统和软件的更新。

因为Ubuntu是一个开源的系统，会不定期的提供一些更新，下载的镜像文件不一定是最新的，这里先更新一下系统。按照下面的操作步骤：

1）打开终端（快捷键：ctrl+alt+t），输入 “sudo apt-get update”，按回车键。输入管理员密码（使用sudo passwd root来设置，sudo命令是给普通用户管理员的权限去操作系统），按回车键。apt-get update 是刷新可安装的软件列表,但是不做任何实际的安装动作,apt-get upgrade是进行安装包的更新，这是软件版本的升级。

2）输入 “sudo apt-get dist-upgrade”，按回车键。进入列新你系统和系统里安装的软件在出现“你希望继续执行嘛？[Y/N]”,请输入“Y”，进入自动更新Ubuntu系统及你安装的软件。

2.除了上面三篇博客说的那些，还要进行一些其他的配置

1）对fireFox的设置。

在没有翻墙之前设置firefox对百度的支持。

i.设置主页为百度，如上图，HomePage填写Baidu的网址。

ii.设置百度为默认的搜索引擎：右边菜单——>preference(选项)——>search——>add more search engine ，翻三页找到百度，然后add to firefox，然后设置百度为默认搜索引擎。

2）安装为之笔记（个人习惯），并添加firefox的为之笔记的插件，轻松截取相关的页面

i.安装为知笔记

从官方PPA仓库安装：

$ sudo add-apt-repository ppa:wiznote-team

$ sudo apt-get update

$ sudo apt-get install wiznote

ii.firefox插件的安装

点击下载并手动安装，下载解压后将.xpi文件拖拽到Firefox窗口内即可。

4）安装wine QQ

i.在Linux 麒麟网站下载winqq：http://www.ubuntukylin.com/application/

ii. 具体安装流程可以参考下面的blog：
http://www.linuxdiyf.com/linux/15428.html

5）在左侧菜单添加“显示桌面”和“工作空间切换器”，“显示桌面”可以在打开页多的情况下方便的回到桌面。“工作空间切换器”可以为不同的工作空间添加任务。使用alt + ctrl +”上下左右键” 来切换。设置如下。

三，神器部分

到了令人激动人心的时候了。

1.护眼神器：f.lux

作为开发人员，长期对着电脑，要对眼睛进行保护。这里使用f.lux,相关安装教程如下：

http://my.oschina.net/u/2003106/blog/374003

安装完毕后，设置经纬度，如上图，注意加上N（北纬）和E（东经），我开始没有设置就调整不对。

2.屏幕取色神器：gcolor2

Android程序员要对一些颜色进行设置，这里使用gcolor2,使用起来也超级简单，点击吸管，然后在屏幕上取色就可以了。取到的颜色就会在面板上显示。安装命令：

sudo apt-get install gcolor2

安装完毕，在终端输入gcolor，点击enter。打开后可以固定到左侧菜单栏。

3.屏幕尺子：screenRuler 在软件中心中下载安装

4.查看合并冲突神器：kdiff3，安装命令：

sudo apt-get install  kdiff3

5.类似与window的画画板工具：kolourpaint4，用它可以在写文档时画一些简单的流程图。

安装：sudo apt-get install kolourpaint4

打开：kolourpaint

6.思维导图工具：FreeMind

从软件中心下载安装，下载完毕后图标会出现在左侧菜单栏，点击启动。
